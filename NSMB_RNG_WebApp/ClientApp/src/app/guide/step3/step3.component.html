The next thing to do is figure out what date and time you'll use to collect RNG data.

RNG results depend on (among other things) the date and time to the second at which RNG was initialized.
This occurs about half a second before the Nintendo logo begins to appear at startup.
You'll need to go to the system settings and set the date and time, to whatever you want.
However, you should always ensure the console's clock shows a different hour/minute when you set those.*
Get a stopwatch with a precision of at least tenths of seconds. Start the stopwatch at the same time you set the system clock.
DO NOT attempt to use the home screen's clock for timing anything. It is not precise enough and can display seconds incorrectly.
After setting the time, start the game. Stop the stopwatch when the Nintendo logo beings to appear.
To get the time at which RNG was initialized, subtract a little over half a second from the time on your stopwatch then round down to the nearest second.

For example, lets say you set the date and time to 08/11/2004 05:00.
If the Ninendo logo appears at 13.2 seconds, subracting half gives 12.7 and then rounding down gives 12.
So you would enter 08/11/2004 05:00:12 below in this case.

You're going to have to set the time to the same value and reload the game multiple times in the next steps.
There are a few other things you should be aware of before proceeding:

  • You should not hold any buttons during the game's start-up sequence.
  • You also should not have a game in the GBA slot. The blank cartridge that comes with the DS is okay.
  • If your game version supports multiple languages, changing the system language changes RNG. (You can use any language.)

*On some consoles, if it sees you've set the time to, for example 3:15 but it was already 3:15 it will not actually do anything. This is bad since you need it to set to 3:15:00.
