Manipulate Random Numbers in NSMB

This web app is intended to aid in manipulating random number generation (RNG) in New Super Mario Bros. (NSMB) on DS-like and 3DS-like consoles. This will allow:

  • Set up only once per gaming session. After that, just go back to main menu and start a new game.
  • Always have red ? blocks move to the left in W8.
  • Always get a blue shell powerup in 1-2 (requires controlling double jumps in 1-1).
  • Have a greatly increased chance of getting a mini mushroom in 5-Ghost (advanced strategy only for top-level players).

Get started setting up here. You can view video instructions for using this website here.
